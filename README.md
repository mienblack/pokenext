## 💻 Projeto

O pokenext é uma pokedex desenvolvida em React você pode visualizar alguns pokemon e suas informações.

## 🚀 Tecnologias

As seguintes ferramentas foram usadas na construção do projeto:

- [NextJS]
- [ReactJs]
- [Javascript]
- [HTML]
- [CSS]

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
